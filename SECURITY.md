# Security Policy

## Reporting a Vulnerability

We take security seriously at Kleros AI. If you discover a security vulnerability, please report it responsibly.

### How to Report

1. **Email:** Send details to **security@klerosai.com**
2. **Subject:** Use `[SECURITY]` prefix in your email subject
3. **Include:**
   - Description of the vulnerability
   - Steps to reproduce
   - Potential impact
   - Any suggested fixes (optional)

### What to Expect

| Timeline | Action |
|----------|--------|
| **24 hours** | Acknowledgment of your report |
| **72 hours** | Initial assessment and severity rating |
| **7 days** | Status update on fix progress |
| **30 days** | Target resolution for critical issues |

### Scope

This policy applies to:
- All repositories under the Kleros-AI organization
- klerosai.com and related web properties
- API endpoints and services

### Safe Harbor

We will not pursue legal action against researchers who:
- Report vulnerabilities responsibly
- Avoid accessing or modifying user data
- Do not disrupt our services
- Allow reasonable time for fixes before disclosure

### Recognition

We appreciate security researchers and may:
- Acknowledge contributors in our security hall of fame
- Provide references for responsible disclosures

*Thank you for helping keep Kleros AI secure.*
